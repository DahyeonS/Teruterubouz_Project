# 개요
**MyBatis 작업을 위한 xml 파일들을 모아놓은 곳입니다.**

## *mybatis-config.xml*
자료형을 정의한 곳입니다.

## mapper
**SQL 쿼리 처리용 Mapper 파일들을 모아놓은 곳입니다.**

### *StatusMapper.xml*
Status 폴더의 Mapper 파일

### *MemberMapper.xml*
Member 폴더의 Mapper 파일

### *BoardMapper.xml*
Board 폴더의 Mapper 파일

### *NoticeMapper.xml*
Notice 폴더의 Mapper 파일
